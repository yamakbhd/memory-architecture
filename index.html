1

项目记忆 Project Memory

最顶层 · 持久化 · 多项目隔离
▼

项目记忆是最顶层的持久化记忆层，负责管理多个独立项目的全生命周期上下文。每个项目拥有独立的命名空间，同时支持跨项目的显式共享声明。

  • 多项目独立管理，项目间完全隔离
  • 跨项目共享记忆（显式声明后复用通用经验）
  • 时间线和优先级管理，支持并行任务调度
  • 通过唯一 project_id 实现标识和隔离
2

规划记忆 Plan Memory

任务编排 · 自动分解 · 依赖检测
▼

规划记忆负责多轮任务规划与执行，支持将高层任务自动分层分解为可执行的子任务，并进行动态调度。

  • 多轮任务规划和执行引擎
  • 自动分层分解为子任务
  • 依赖检测和动态调优
  • 优先级冲突的自动解决
3

对话记忆 Session Memory

四层嵌套 · 有损压缩 · 跨会话
▼

对话记忆通过四层嵌套结构管理当前会话的信息流。从长期知识库到当前任务上下文，逐层压缩以适应有限的上下文窗口。

  • 长期记忆：知识库 · 经验库 · 行为库
  • 短期记忆：完整对话历史 · 用户信息 · 工具信息
  • 有损短期记忆：上轮规划阶段对话摘要 + 代码变更
  • 当前任务记忆：代码片段 · 最近行为 · 步骤序列
4

事件记忆 Event Memory

事件循环 · 交互日志 · 工具注册
▼

事件记忆记录 Agent 的实时行为轨迹，包括事件循环序列、交互日志、任务状态变迁和工具调用注册信息。

  • 事件循环序列记录
  • 完整的交互日志
  • 任务状态变化追踪
  • 工具调用注册与生命周期管理
5

训练接口 Training Interface

在线学习 · 离线调优 · 反馈闭环
▼

训练接口提供从运行时行为到模型优化的完整反馈闭环。支持在线学习管线和离线调优基准。

  • 在线学习管线：实时反馈驱动改进
  • 离线调优基准：批量训练数据管理
  • 行为-结果配对记录
  • A/B 测试与模型版本管理
